# 大神燒肉材料管理 ERP v3.9 Mega

本版將 v3.5、v3.6、v3.8、v3.9 合併為一包升級版，保留既有 v3.4 手機快速 Key-in、員工權限、採購編輯／作廢功能，並加入下一階段採購管理功能。

## 本版整合內容

### v3.5 供應商管理與價格追蹤
- 新增「供應商管理」頁
- 供應商主檔可新增／更新
- 顯示主要品項、本月採購、累計採購、最後採購日
- 可依歷史採購資料自動彙整供應商表現

### v3.6 標準品項管理
- 新增「品項管理」頁
- 可設定標準品項名稱、品項別名、預設供應商、預設單位、換算倍率
- 用來統一不同員工輸入的品項名稱，避免成本分析失真

### v3.8 月報表與成本分析
- 新增「月報表」頁
- 可依月份查看總採購、食材成本、酒水成本、最大分類
- 顯示分類排行、供應商排行、品項價格漲跌提醒
- 可匯出該月份採購 CSV

### v3.9 AI 待確認流程
- AI 單據辨識不直接入帳
- AI 辨識後會進入「AI待確認」頁
- 人工確認後才批次寫入採購紀錄
- 報價單紀錄會保留 AI 原始 JSON、確認人、確認時間與確認筆數

## 更新方式

### 1. 覆蓋 GitHub 前端
把以下檔案覆蓋到 GitHub Pages 專案根目錄：

```text
index.html
app.js
style.css
data.js
README.md
manifest.webmanifest
```

### 2. 覆蓋 Apps Script 後端
把 `google-apps-script/Code.gs` 全部貼到 Apps Script，覆蓋原本內容。

部署時請務必：

```text
部署 → 管理部署作業 → 鉛筆 → 新版本 → 部署
```

### 3. Google Sheet 欄位
本版會自動補齊新增欄位。主要新增：

材料主檔：品項別名、預設供應商、預設分類、預設單位、預設換算倍率、預設標準單位、合併到標準品項、價格警戒百分比。

供應商主檔：Email、統一編號、供應商類型、預設付款天數、最後採購日、本月採購金額、累計採購金額、價格穩定度、推薦狀態。

報價單紀錄：AI原始JSON、確認狀態、確認筆數、錯誤訊息。

## 測試建議

1. 用 boss 登入。
2. 開啟供應商管理，新增一筆供應商。
3. 開啟品項管理，新增或編輯一筆標準品項。
4. 開啟月報表，選擇本月，確認分類與供應商排行。
5. 上傳測試單據，確認 AI 結果進入 AI待確認，不會直接入帳。
6. 在 AI待確認中人工確認並寫入採購紀錄。
