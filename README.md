# 大神燒肉材料管理 ERP v3.9.2 Sidebar Overlap Fix

本版為 v3.9 Mega 的左側選單修正版。

## 修正內容

1. 桌機版左側選單改成三段式：
   - 上方品牌區固定
   - 中間功能選單可上下滑動
   - 下方登入者資訊與登出按鈕固定
2. 側邊欄登入者區只顯示角色，不再顯示整串權限文字，避免文字過長造成重疊。
3. 不需要修改 Apps Script，也不需要重新部署後端。

## 更新方式

只需覆蓋 GitHub 前端：

- app.js
- style.css

若你想讓版本完整一致，也可一併覆蓋其他前端檔案，但本修正只需要上述兩個檔案。

更新後請開：

https://guessman0509.github.io/okami-material-erp/?v=392

並按 Ctrl + F5 強制重新整理。
